## 📊 The Problem

**Manual payroll data extraction is time-consuming and error-prone:**

- ⏰ Takes **2 hours** per execution
- 👤 Requires dedicated staff time
- ❌ Prone to **copy-paste errors**
- 😓 Tedious and repetitive work
- 📉 Not scalable as data grows

---

## ✅ The Solution

**Automated payroll data extraction using Playwright:**

A one-command automation that extracts Paycode and Earning Code data from all 15 calculation groups and generates a formatted Excel report.
